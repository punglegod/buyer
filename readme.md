## 📖 简介

**采购订单助手**是一款面向采购人员的桌面效率工具，能够将 ERP 系统复制出的采购订单数据**自动按供应商拆分**，并基于 Excel 模板**一键生成**规范的抛单文件。

告别手动复制粘贴、逐个填写表格的繁琐工作，只需一次粘贴，即可批量生成所有供应商的抛单文件。

## ✨ 核心功能

| 功能 | 说明 |
|------|------|
| 🔀 **智能拆单** | 自动识别并按供应商分组，每个供应商生成独立 Excel 文件 |
| 📋 **剪贴板直读** | 输入框为空时，自动读取系统剪贴板内容，复制即用 |
| 📐 **行高自适应** | 根据物料描述长度自动计算行高，中英文混排精确适配 |
| 🏷️ **智能缩写** | 自动从供应商全称提取简称，过滤地名/行业词，支持手动映射 |
| 🔄 **找回输入** | 拆单成功后自动清除输入，支持一键找回上次数据 |
| 📁 **一键打开** | 快速打开输出文件夹查看生成的文件 |
| 🎨 **精美界面** | iOS 26 Liquid Glass 风格毛玻璃 GUI，多段渐变背景 |

## 🚀 快速开始

### 环境要求

- Python 3.8+
- 依赖库：`openpyxl`

### 安装

```bash
git clone https://github.com/punglegod/buyer.git
cd buyer
pip install openpyxl
```

### 运行

```bash
python main.py
```

### 使用流程

1. 从 ERP 系统复制采购订单数据（支持 Tab 分隔格式）
2. 粘贴到输入框，或直接点击「一键拆单生成」自动读取剪贴板
3. 程序自动按供应商拆分并生成 Excel 文件
4. 点击「打开输出文件夹」查看生成的文件

## ⚙️ 配置说明

### 供应商映射表

在 `main.py` 顶部的 `SUPPLIER_MAPPING` 字典中，可以手动配置供应商全称到简称的映射：

```python
SUPPLIER_MAPPING = {
    "xx科技有限公司": "xx",
    "xx股份有限公司": "xx",
    # ... 添加更多映射
}
```

未在映射表中的供应商将自动提取简称（去除地名、行业词、公司后缀，取前 4 个字符）。

### 地名过滤

程序内置了全国省份、直辖市及主要城市的名称列表，并**自动生成带"市"的变体**（如 `嘉兴` → `嘉兴` + `嘉兴市`），支持精确匹配不同格式的公司全称。

如需新增城市，只需在 `_base_places` 列表中添加即可。

## 📝 数据格式

程序支持两种输入格式：

### Tab 分隔格式（推荐）

直接从 ERP 系统复制的数据，字段以 Tab 分隔：

```
部门  人名  日期  订单号  物料代码  物料描述  状态  单位  数量  交期  供应商
```

### 空格分隔格式（兼容）

自动通过正则匹配提取关键字段，兼容格式不规范的数据。
